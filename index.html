Methodology

A four-stage workflow from raw survey data to a prioritized recommendation.

01

Data cleaning

Normality checks and pre-processing performed in R on the raw survey export.

02

Model building

Reflective constructs (motivation, engagement, outcomes) and a formative construct (gamification elements) configured as a path diagram in SmartPLS.

03

Validation

Bootstrapping (N = 5,000) for path significance; Blindfolding (D = 7) for predictive relevance.

04

Post-hoc analysis

IPMA run on the structural model to separate what matters from what's merely visible.

Key findings

Motivation → Engagement

Intrinsic motivation is a strong, significant predictor of engagement (β = .41, p < .05).

Engagement → Outcomes

Engagement is the strongest driver in the model (β = .52), mediating nearly all downstream effect on learning outcomes.

Priority signal

Feedback speed shows high importance and low performance — the single highest-leverage fix available.

Over-invested signal

Leaderboards are visible but low-impact; competitive display elements are not where learning actually happens.
